# SpeakAI

SpeakAI is an English speaking practice system by scenario, built with:

- Frontend: React + Vite + Firebase Auth
- Backend: FastAPI + MongoDB + ChromaDB
- LLM: Groq
- STT: Groq Whisper
- TTS: edge-tts

## Main Features

- Firebase login
- Scenario browsing
- Practice session creation
- Text chat with AI
- Voice turn: audio -> STT -> RAG -> LLM
- AI reply TTS
- Conversation history
- Session feedback
- AI response reporting
- Notifications
- Progress summary
- Admin analytics
